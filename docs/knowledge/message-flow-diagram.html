Message Flow

Rough end-to-end path for one inbound message through Manager AI, from transport and thread loading to workflow decisions, persistence, and outbound reply delivery. 

flowchart TD
    A["Inbound message
phone + text"] --> B["FastAPI route
api/routes.py"]
    B --> C["App wiring builds agent from TOML config
api/main.py + src/manager_ai/wiring"]
    C --> D["Agent.handle_message()
creates IncomingMessage"]
    D --> E["Load or create thread
ContactThreadState"]
    E --> F["Normalize and append inbound message
record incoming_message event"]
    F --> G["Classify intent
MessageClassifierPort"]
    G --> H["Select route
record intent_detected and route_selected"]
    H --> I{"Reuse active job
or create new job?"}
    I -->|Reuse| J["Use existing JobState
record job_selected"]
    I -->|Create| K["Create new JobState via thread_router
record job_created"]
    J --> L{"Is this an En Red service request?"}
    K --> L
    L -->|No| M["Mark job disqualified
append not-qualified reply"]
    L -->|Yes| N["Run structured extraction
update job data"]
    N --> O["Recompute missing fields and evidence status
record extraction/missing-field events"]
    O --> P["Quote handling
ensure_quote()"]
    P --> Q["Scheduling handling
handle_scheduling()"]
    Q --> R["Closure and follow-up handling
apply_closure_updates()"]
    R --> S{"Any outbound reply already prepared?"}
    M --> T["Append outbound message(s) to history
record outbound/status events"]
    S -->|Yes| T
    S -->|No| U{"Job status after processing?"}
    U -->|scoping| V["Promote to estimate_ready
reply: ready for quote/review"]
    U -->|awaiting_evidence| W["Reply with next missing-field question"]
    U -->|Other| X["Generate fallback conversational reply"]
    V --> T
    W --> T
    X --> T
    T --> Y["Create escalation actions if needed"]
    Y --> Z["Persist updated thread/job
via repository or storage adapter"]
    Z --> AA["Send outbound message(s)
via messaging adapter"]
    Z --> AB["Return workflow result
with thread, outbound messages, external actions"]
