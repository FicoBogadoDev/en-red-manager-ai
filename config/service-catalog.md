# Catalogo de servicios de En Red Rosario

Este archivo lo puede editar el equipo de En Red. Escribi cada servicio o caso
como una linea con guion. Si queres definir una respuesta corta para algo que no
ofrecemos, agregala en la misma linea despues de `Respuesta:`.

## Si ofrecemos

- Redes de seguridad para balcones, terrazas, techos y escaleras.
- Redes de proteccion para chicos, gatos y mascotas.
- Otra red de seguridad para otro ambiente, departamento, depto o direccion.

## No ofrecemos, pero puede aparecer junto a una consulta valida

- Mosquiteros. Respuesta: No instalamos mosquiteros.
- Cerramientos de vidrio o blindex. Respuesta: No hacemos cerramientos de vidrio.
- Arreglos o reparaciones de barandas. Respuesta: No hacemos arreglos de barandas.
- Pintura. Respuesta: No hacemos trabajos de pintura.
- Obra, albanileria, revoques o durlock. Respuesta: No hacemos trabajos de obra general.

## Claramente fuera de rubro

- Redes de pesca. Respuesta: No trabajamos redes de pesca.
- Redes deportivas de futbol o volley. Respuesta: No trabajamos redes deportivas.
- Redes informaticas, red LAN, wifi o cableado de red. Respuesta: No trabajamos redes informaticas.

## Dudoso: preguntar antes de rechazar

- Proteccion para mascotas sin mencionar red, balcon, terraza, techo o escalera.
- Proteccion en terraza sin aclarar si busca red de seguridad u otro cerramiento.
